## How to request data

Submit all data-use requests via the Shepherd Research Lab Analysis Plan Form:

[Submit an Analysis Plan →]({{ site.baseurl }}/request-data/){: .btn .btn-primary}

Before requesting, please review the **Hawaiʻi & Pacific Islands Mammography Registry Data Use Agreement** (template — final agreements may vary by institution). Contact the lab if you need a copy.

## Requirements

- IRB approval is required for all data use
- Investigators must submit timely reports on data use
- Publications from data analysis must acknowledge the funding sources for HIPIMR
